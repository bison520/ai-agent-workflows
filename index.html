新手入門指南

01

N8N 是什麼？一分鐘搞懂自動化工作流

N8N 是一個開源自動化平台，讓你用視覺化介面串接各種服務（Gmail、LINE、蝦皮、ChatGPT 等），建立自動化流程。不需要寫程式，只要拖拉節點就能完成。

入門概念 3 分鐘閱讀
02

如何匯入 JSON 工作流？三步驟完成

從本站下載 JSON 檔案後，開啟 N8N 點選「Import from File」，選擇下載的 JSON 檔案即可匯入。匯入後填入你的 API 金鑰，工作流就能開始運作。

實作教學 5 分鐘完成
03

如何用精準提示詞省 Token？

模糊的提示詞會讓 AI 反覆試錯，浪費 Token。本站每個工作流都附帶精準的中文提示詞，直接複製貼上就能大幅減少 API 呼叫次數。
