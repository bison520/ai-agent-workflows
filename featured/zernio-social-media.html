🤔 Zernio 是什麼？

Zernio 是一個跨平台社群媒體管理工具，讓你用一個 API 同時發布貼文到 Facebook 粉絲專頁、Instagram 和 TikTok。對於一人公司、小編、自媒體創作者來說，它是省時利器。

💡 免費方案亮點

一個電子郵件帳號可以免費綁定兩組社群帳號。例如你可以綁定一個 Facebook 粉專 + 一個 Instagram 帳號，或是兩個不同的 FB 粉專，完全免費。如果需要管理更多帳號，再考慮升級付費方案即可。

🚀 4 步驟快速上手

1

註冊帳號

前往 zernio.com 註冊，完成 Email 驗證即可。

2

取得 API Key

進入 Dashboard → Settings → API Keys，點擊 Generate New Key，複製並妥善保存（只會顯示一次）。

3

綁定社群帳號

進入 Dashboard → Accounts，依序連接 Facebook 粉絲專頁、Instagram、TikTok。免費方案可綁定兩組社群帳號。

4

驗證 & 開始發文

執行 auth:set → auth:check → accounts:list 確認連線，然後就可以開始發布貼文了！

📝 發文 API 範例

POST /posts
{
  "content": "你的貼文內容",
  "accounts": ["帳號A_ID", "帳號B_ID"],
  "mediaUrl": "圖片或影片網址（選填）",
  "publish": true
}

在 accounts 陣列中填入多個帳號 ID，就能一次發布到多個平台。

⚠️ 發文鐵則

  1. 段落之間一定要加空行 — 不加空行的話，IG 和 FB 貼文會全部擠在一起
  2. 發文前強制流程 — auth:set → auth:check → accounts:list → 發文
  3. 不同帳號用不同 Key — 切勿用 A 粉專的 Key 發 B 粉專的內容
  4. 來源標註格式 — @影片來源：B站 頻道名稱（不貼連結）

📊 各平台限制

平台支援內容注意事項
Facebook文字、圖片、影片支援留言功能，可發布到粉絲專頁
Instagram圖片、影片（Reels）⚠️ 必須有圖片或影片才能發布
TikTok影片發布❌ 不支援留言，分潤連結需放在影片描述中

📋 發文檢查清單

  • API Key 已設定（auth:set）
  • 連線狀態正常（auth:check）
  • 帳號清單正確（accounts:list）
  • 貼文內容段落間有空行
  • 來源標註格式正確
  • 圖片/影片符合平台規格
